# NeuroPlan AI — Project Manifest

## 🎯 Vision
Transform study planning from a static calendar into an autonomous execution agent that decomposes complex curricula, builds intelligent study material, and adapts schedules in real-time based on verified mastery.

## 🏗️ Core Pillars
1. **Curriculum Autonomy**: AI-powered subject decomposition into dependency-aware subtopics.
2. **Content Intelligence**: Spaced-repetition aware study cards and active knowledge probing.
3. **Elastic Execution**: Constraint-based scheduling that adapts to real-world deviations.

## 🛠️ Stack
- **Backend**: FastAPI, PostgreSQL, SQLAlchemy 2.0, OpenAI/Claude
- **Frontend**: React 18, Vite, TailwindCSS, Zustand
- **Orchestration**: Docker Compose
